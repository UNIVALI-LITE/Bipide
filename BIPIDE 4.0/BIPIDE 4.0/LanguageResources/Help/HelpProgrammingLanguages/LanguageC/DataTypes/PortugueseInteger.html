Tipo int

São aceitos pelo C do BIPIDE apenas os tipos de dados ints sendo estes de 32 bits constituidos dos números naturais positivos {1,2,3, ...}, negativos {-1, -2, -3, ...} e zero {0}.

O tipo de dado int pode ser representado nas notações decimal, hexadecimal e binária.

Exemplo:


    int main()
    {
        //imprime o número int 32
        //escrito em notação decimal
        printf(32)

        //imprime o numero int 32 
        //escrito em notação hexadecimal
        printf(0x20)

        //imprime o número int 32
        //escrito em notação binária
        printf(0b00100000)
		
		return 0;
    }
