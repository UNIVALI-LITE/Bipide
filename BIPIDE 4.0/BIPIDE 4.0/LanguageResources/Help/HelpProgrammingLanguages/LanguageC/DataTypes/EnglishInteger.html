'int' Type

Are accepted by the C BIPIDE only integer data types and these constituted 32 bits of positive natural numbers {1,2,3, ...}, negative {-1, -2, -3, ...} and zero {0}.

The 'int' data type can be represented in decimal, hexadecimal and binary notation.

Example:

		int main()
		{
			//prints the integer 32
			//written in decimal notation
			printf(32)

			//imprime o numero int 32 
			//written in hexadecimal notation
			printf(0x20)

			//prints the integer 32
        //written in binary notation
			printf(0b00100000)
			
			return 0;
		}
