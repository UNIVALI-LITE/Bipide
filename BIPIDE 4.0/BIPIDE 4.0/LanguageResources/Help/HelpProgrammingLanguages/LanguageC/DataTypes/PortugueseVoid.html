Tipo void

Void é usado para o resultado de uma função que não retorna nenhum valor.

Normalmente, as funções de tipo void são chamadas por seus efeitos como, por exemplo, a realização de alguma tarefa ou escrevendo os seus parâmetros na saída de dados.

A função com o tipo void termina ou por atingir o final da função ou executando um comando retorne sem valor retornado.

Exemplo:


    int main()
    {
        imprime(1)
        imprime(2)
        imprime(3)

    }

    //Função de retorno void que escreve variavel
    int void imprime(int var) 
    {
        printf(var)
    }
