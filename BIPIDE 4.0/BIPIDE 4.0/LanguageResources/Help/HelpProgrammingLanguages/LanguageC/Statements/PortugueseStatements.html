Declarações

Uma declaração especifica o identificador, tipo, e outros aspectos de elementos da linguagem, tais como variáveis ​​e funções. É utilizado para anunciar a existência do elemento para o compilador

Para as variáveis​​, a declaração reserva uma área de memória para armazenar valores. Para as funções, declarações fornecem o corpo e assinatura da função.

Enquanto uma função ou variável pode ser utilizada muitas vezes, ela é declarada, apenas uma vez.

Nesta seção serão apresentadas as variadas formas de declaração que o C possui.
