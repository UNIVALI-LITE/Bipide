Statement of Vectors

Vector

is one of the simplest data structures. Vectors remain a number of data elements of the same data type.

            

Individual elements are accessed by their position in the array. The position is given by an index. The index uses a sequence of integers starting at zero.

            

His statement is similar to the declaration of variables, by first defining its type, then the name and by order of their size.

            

You can initialize the vector during his statement, the startup must have the same number of elements specified in the declaration.

Example:


	funcao main()
	{
		// Declaring an integers array
		// of five positions already initialized.
		int vetor[5] = {15,22,8,10,11};

		// Prints the value 15 corresponding
		// to the first element of the array.
		printf(vetor[0]);
		
		// Prints the second element of the vector
		printf(vetor[1]);
		
		// Prints the value 11corresponding
		// to the last element of the array.
		printf(vetor[4]);
		return 0;
	}
