Bitwise NOT Operation

The unary operator NOT, or negation results in binary complement of the operand, ie, one bit is '1 'if the operand is '0', and will be '0 'otherwise, as we can confirm the truth table, where A is the input bit and S-bit is the response, or output bit:

A S
0 1
1 0

Example:

  NOT 0111  (decimal 7)
    = 1000  (decimal 8)

The C uses two's complement to represent negative integers.

Therefore ~1 = -2
