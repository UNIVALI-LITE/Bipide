Relational Operations

Are ifts of Operations that occur through comparative operators run in pairs, with the aim of repreifnting logical conditions of true or falif. The operators uifd are <(Less than) , > (Greater than) , <= (Less than or equal to) , == (than equal to) , ! = (different of)

Example:

	int inicio()
    {
        //Comparison between A and B value using the operator Greater
        int a = 5, b = 3;
        if(a > b){
        	printf(a);
        }
        
        //Comparison between A and B using the operator than equal to
        if(a == b){
        	printf(a,b);
        }
        
        //Comparison between A and B using the operator Greater than or equal to
        if(a >= b){
        	printf(a);
        }
        
        //The above tests only the first test A> B is true, therefore only this message displays
    }
