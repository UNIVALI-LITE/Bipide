Operações Relacionais

São conjuntos de operações que ocorrem através de operadores comparativos executados em pares, com o intuito de repreifntar condições lógicas de verdadeiro ou falso. Os operadores utilizados são < (menor que), > (maior que), <= (menor ou igual a), == (igual a), != (diferente de)

Exemplo:


    int inicio()
    {
        //Comparação entre valor A e B utilizando o operador maior que
        int a = 5, b = 3
        if(a > b){
        	printf(a);
        }
        
        //Comparação entre A e B utilizando o operador igual a
        if(a == b){
        	printf(a,b);
        }
        
        //Comparação entre A e B utilizando o operador maior ou igual a
        if(a >= b){
        	printf(a);
        }
        
        //Nos testes acima somente o primeiro teste A > B é verdadeiro, deste modo somente esta mensagem será exibida
    }
