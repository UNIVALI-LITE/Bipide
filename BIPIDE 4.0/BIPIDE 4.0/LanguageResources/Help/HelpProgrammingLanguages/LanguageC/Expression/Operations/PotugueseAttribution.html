Atribuições

Consiste em utilizar um valor obtido através da entrada de dados se já informado pelo usuário ou por através de uma constante, de modo que este valor se já armazenado por uma variável declarada ou um espaço de memória

Exemplo:


    int main()
    {
        //Atribuição de valores constantes a uma variável
        int a;
        a = 2;
        
        //Atribuição através de entrada de dados, informado pelo usuário
        int b;
        scanf(b);
        
        //Atribuição através de uma variável já informada pelo usuário
        int c;
        c = b;
    }
