Addition operation

Addition is one of the basic operations of algebra. In its simplest form, addition combines two numbers (terms somandos or parcels) in a single number, the sum or total. Add more numbers corresponds to repeat the operation.

Propriedades importantes

  • Commutative

    The order of the plots does not change the result of the operation. Thus, if 2 + 3 = 5, soon 3 + 2 = 5.

  • Associativity

    The grouping of the plots does not change the result. Thus, if (2 + 3) + 1 = 6, soon 2 + (3 + 1) = 6.

  • Neutral element

    The portion 0 (zero) does not change the outcome of the other installments. The zero is called "neutral element" of the addition. Thus, if 2 + 3 = 5, soon 2 + 3 + 0 = 5.

  • Annulment

    The sum of any number and its opposite is zero. Example:

    • 2 + (-2) = 0
    • (-999) + 999 = 0
