if

The if command is used to divert the execution of the program based on                 a condition.

This command has a logical expessão to be evaluated. If this expression is                 true a specific list of commands will be executed, if false, a list                 Alternatively the command will be executed (if any). To define the list of commands that                 will be executed if the condition is not met, you must use the reserved word                 'senao'.

example:

		int main()
        {
            int estudo = 1;
            int aprendo;
            int ficoInteligente;
            int ficoBurro;
  
            if (estudo == 1)
            {
                 aprendo = 1;
  
                 if (aprendo == 1)
                 {
                      ficoInteligente = 1;
                      ficoBurro = 0;
                 }
                 else
                 {
                     ficoInteligente = 0;
                     ficoBurro = 1;
                 }
            }
            else
            {
                 aprendo = 0;
                 ficoInteligente = 0;
                 ficoBurro = 1;
            }
			return 0;  
		}
