Do While loop (Post-Tested)

This loopping structure the programmer executes a block of statements D and then do a check B if the test condition is true executes the statement block until the condition B is false.

Example:


    int main()
    {
        //Writes to the console the value of the variable pitch during executions
        int passo = 0;
        do{
        	printf(passo);
        	passo = passo + 1;
        }while(passo < 5);

		passo = 0
        //Writes to the console the step value multiplied by 5
        do{
        	printf(passo);
        	passo = passo + 1;
        }while(passo < 10);
		return 0;        
    }
