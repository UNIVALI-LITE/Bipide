printf

The 'printf' command is used when you want to show information in the IDE console, ie it is an output command data.

To use the 'printf' command, you must type this command and put in parenthesis (s) variable (s) you want to display on the console.

Example:

		
	int main()
    {
        //prints the integer 32
        printf(32);

        int a = 15;
        //writes to the console the value of variable "a"
        printf(a);

    }
