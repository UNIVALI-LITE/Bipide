scanf

The 'scanf' command is used when you want to get information from the computer keyboard, in other words, is a command input. This command waits for a value to be entered directly into the variable and assigns.

To use the 'scanf' command, you must type this command and put in parenthesis (s) variable (s) you want to receive the amounts to be entered.

Example:

	
	int main()
    {
        int numero;
        
        //reads the value entered by the user
        scanf(numero);

    }
