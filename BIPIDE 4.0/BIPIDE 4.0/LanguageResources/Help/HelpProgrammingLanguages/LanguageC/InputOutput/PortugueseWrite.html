printf

O comando printf é utilizado quando deseja-se mostrar informações no console da IDE, ou seja, é um comando de saída de dados.

Para utilizar o comando printf, você deverá escrever este comando e entre parênteses colocar a(s) variavel(eis) que você quer mostrar no console.

Exemplo:


    int main()
    {
        //imprime o número int 32
        printf(32);

        int a = 15;
        //escreve no console o valor da variável "a"
        printf(a);

    }
