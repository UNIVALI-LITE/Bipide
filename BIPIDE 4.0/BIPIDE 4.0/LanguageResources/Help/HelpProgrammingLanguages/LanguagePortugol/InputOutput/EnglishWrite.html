Escreva

The 'escreva' command is used when you want to show information in the IDE console, ie it is an output command data.

To use the 'escreva' command, you must type this command and put in parenthesis (s) variable (s) you want to display on the console.

Example:

programa
{
    funcao inicio()
    {
        //prints the integer 32
        escreva(32)

        inteiro a = 15
        //writes to the console the value of variable "a"
        escreva(a)

    }
}
