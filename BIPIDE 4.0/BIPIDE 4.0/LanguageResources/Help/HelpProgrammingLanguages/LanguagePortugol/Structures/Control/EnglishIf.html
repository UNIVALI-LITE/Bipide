Se

The 'se' command is used to divert the execution of the program based on                 a condition.

This command has a logical expessão to be evaluated. If this expression is                 true a specific list of commands will be executed, if false, a list                 Alternatively the command will be executed (if any). To define the list of commands that                 will be executed if the condition is not met, you must use the reserved word                 'Senao'.

example:

programa
{
        funcao inicio()
        {
            inteiro estudo = 1
            inteiro aprendo
            inteiro ficoInteligente
            inteiro ficoBurro
  
            se (estudo == 1)
            {
                 aprendo = 1
  
                 se (aprendo == 1)
                 {
                      ficoInteligente = 1
                      ficoBurro = 0
                 }
                 senao
                 {
                     ficoInteligente = 0
                     ficoBurro = 1
                 }
            }
            senao
            {
                 aprendo = 0
                 ficoInteligente = 0
                 ficoBurro = 1
            }
  
	}
}
