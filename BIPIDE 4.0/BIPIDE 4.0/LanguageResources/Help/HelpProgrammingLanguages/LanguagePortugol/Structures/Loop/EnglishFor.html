For loop (With Control Variable)

This looping structure the programmer must first declare a variable control A to be "tested" according to a condition B if that condition is true, the internal code within the structure will be executed. At the end of the variable declared in will be incremented or decremented and then the condition B will again be "tested" until it is false.

Example:

programa
{
    funcao inicio()
    {
        //Writes to the console the value of the variable pitch during executions
        para(inteiro passo = 0; passo < 10 ; passo = passo + 1){
        	escreva(passo)
        }

        //Writes the values ​​to the console 10 a 1
        para(inteiro passo = 10; passo >= 1; passo = passo - 1){
        	escreva(passo)
        }
    }
}
