While loop (Pre-Tested)

This repeating structure the programmer must define a condition that will be checked at each iteration of the loop B if the condition is true then runs the block of code D and then control variable must be changed to prevent the repeat loop is executed infinitely.

Example:

programa
{
    funcao inicio()
    {
        //Writes to the console the value of the variable pitch during executions
        inteiro passo = 0
        enquanto(passo < 5){
        	escreva(passo)
        	passo = passo + 1
        }

		passo = 0
        //Writes to the console the step value multiplied by 5
        enquanto (passo < 10){
        	escreva(passo*5)
        	passo = passo + 1
        }

    }
}
