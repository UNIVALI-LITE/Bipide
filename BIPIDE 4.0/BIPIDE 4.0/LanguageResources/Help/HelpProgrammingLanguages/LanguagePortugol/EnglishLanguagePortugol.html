Portugol Language:

A programming language is a standardized method for communicating instructions to a computer. It is a set of syntactic and semantic rules used to define a computer program. Allows a programmer to specify precisely which data on a computer will operate, how they will be stored or transmitted, and what actions should be taken under various circumstances.

The Portugol is a representation that is very similar to the C language, is written in Portuguese though. The idea is to facilitate the construction and reading of algorithms using an easier language students.

General Structure of a Program:

A program shall have a main scope with name 'programa'.

Inside the main scope is also compulsory to declare the main function called as 'inicio'

example:

//The 'programa' command is mandatory
programa
{
    /* 
     *  Within the program is allowed to 
	 *  declare global variables, global 
	 *  constants and functions in any order.
     */

    //Function declarations only
    //are permitted inside the program.
    funcao inicio()
    {
        /*
         * Statements of local variables,
         * local constants, structures
         * of control and expressions.
         */
    }
}
