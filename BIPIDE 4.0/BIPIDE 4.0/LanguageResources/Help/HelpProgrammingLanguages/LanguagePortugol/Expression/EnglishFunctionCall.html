Function Call

A Function Call is to run a piece of code decoupled from the rest of the program, it is not necessary to perform steps that are inconsistent with the duties of the procedure. A Function Call or may not return a value and can receive parameters.

Example:

            
programa{
	//function 'soma' receives as a parameter two integers that are reported in function start
	funcao inteiro soma(inteiro a, inteiro b){
		retorne a+b
	}

	//function 'exibir' receives two integer parameters and writes the sum of these values ​​in the console output
	funcao vazio exibir(inteiro a, inteiro b){
		escreva(a+b)
	}

	//function 'mostrar' displays the sum of two values ​​entered by the user
	funcao vazio mostrar(){
		inteiro a, b
		leia(a)
		leia(b)
		escreva(a+b)
	}

    funcao inicio(){
        //Calls the 'soma' function, parameter passing by 5 e 2
        escreva(soma(5,2))
		
		//Calls the 'exibir' function, parameter passing by 5:02
		exibir(5,2)
		
		//Calls the 'mostrar' function, without passing parameter
		mostrar()
    }
}
