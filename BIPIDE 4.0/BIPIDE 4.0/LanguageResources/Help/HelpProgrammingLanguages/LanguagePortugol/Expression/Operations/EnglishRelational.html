Relational Operations

Are sets of Operations that occur through comparative operators run in pairs, with the aim of representing logical conditions of true or false. The operators used are <(Less than) , > (Greater than) , <= (Less than or equal to) , == (than equal to) , ! = (different of)

Example:

programa
{
    funcao inicio()
    {
        //Comparison between A and B value using the operator Greater
        inteiro a = 5, b = 3
        se(a > b){
        	escreva(a)
        }
        
        //Comparison between A and B using the operator than equal to
        se(a == b){
        	escreva(a,b)
        }
        
        //Comparison between A and B using the operator Greater than or equal to
        se(a >= b){
        	escreva(a)
        }
        
        //The above tests only the first test A> B is true, therefore only this message displays
    }
}
