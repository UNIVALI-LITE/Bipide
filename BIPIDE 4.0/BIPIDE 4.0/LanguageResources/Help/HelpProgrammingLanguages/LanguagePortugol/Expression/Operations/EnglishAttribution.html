Assignment

Is to use a value obtained through the data entry to be entered by the user or by a constant, so that this value is stored for a declared variable or memory space

Example:

programa
{
    funcao inicio()
    {
        //Assigning constant values ​​to a variable
        inteiro a
        a = 2
        
        //Assignment through data entry, the user entered
        inteiro b
        leia(b)
        
        //Allocation through a variable already informed user
        inteiro c
        c = b
    }
}
