Declaration of Constants

Constant is an identifier whose associated value can not be changed by the program during its execution.

To declare a constant just add the reserved word const before the declaration of variable, vector or array with initialization.

Example:

programa
{
    //Global constant of 'real' data type
    const real aceleracao_gravidade = 9.78

    funcao inicio()
    {
        //Constant vector location of 'inteiro' data type
        const inteiro teclado_numerico[] = {1,2,3}
    }
}
