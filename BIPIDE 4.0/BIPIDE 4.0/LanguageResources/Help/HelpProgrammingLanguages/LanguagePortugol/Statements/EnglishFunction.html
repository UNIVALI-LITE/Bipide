Statement Functions

The function consists of a piece of code that solves a very specific problem, part of a larger problem.

Some of the advantages of using functions during programming are:

  • the reduction of duplicated code in a program;
  • the possibility of reuse the same code without major changes to other programs;
  • decomposition of the major problems in small pieces;
  • improve the visual interpretation of a program;
  • hide or regular a part of the program, keeping the remaining oblivious to the internal code issues resolved within that function;

components of functions are : < / p >             

                    
  • its prototype , which includes the parameters that are passed to the function at the time of invocation

  • body, which contains the block of code that solves the proposed problem

  • A possible return value , which can be used immediately following the function invocation .

function declaration in Portugol is performed as follows : the reserved word function , followed by the return type , when the return type should be used is concealed the Portugol assumes that the return type is empty , then the function name followed by parentheses opens , a list of parameters can be included before the closing parenthesis to complete the declaration must create the function body.

            

The function body consists of structures within the open and close braces.                When a function possessing a different kind of empty return is obligatory presence of the command to return the body of the function.

            

The declaration of parameters is similar to variable declaration , vector and array without initialization and must be separated by commas .
