'Inteiro' Type

Are accepted by the Portugol BIPIDE only integer data types and these constituted 32 bits of positive natural numbers {1,2,3, ...}, negative {-1, -2, -3, ...} and zero {0}.

The 'inteiro' data type can be represented in decimal, hexadecimal and binary notation.

Example:

programa
{
    funcao inicio()
    {
        //prints the integer 32
        //written in decimal notation
        escreva(32)
        escreva("\n")

        //imprime o numero inteiro 32 
        //written in hexadecimal notation
        escreva(0x20)
        escreva("\n")

        //prints the integer 32
        //written in binary notation
        escreva(0b00100000)
        escreva("\n")
    }
}
