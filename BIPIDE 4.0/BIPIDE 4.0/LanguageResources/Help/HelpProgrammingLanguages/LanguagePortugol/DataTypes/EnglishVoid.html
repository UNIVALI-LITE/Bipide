'Vazio' Type

'Vazio' is used to empty the result of a function that returns no value.

            

Normally, the functions of vacuum type are called by their effects, for Example, performing some task or writing their parameters in the output.

            

Function with the empty type ends or reaches the end of the function or running a command return no return value.

Example:

programa
{	
    funcao inicio()
    {
        imprime(1)
        imprime(2)
        imprime(3)

    }

    //Empty function return that writes variable
    funcao vazio imprime(inteiro var) 
    {
        escreva(var)
    }

}
