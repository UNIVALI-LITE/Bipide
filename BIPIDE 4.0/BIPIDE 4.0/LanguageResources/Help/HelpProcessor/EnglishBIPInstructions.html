Control

HLT Halts
Syntax: HLT
Description: Disables the PC update
Opcode: 00000
Status: No bits affected
PC: It is not updated
Operation: No operation is performed
Example:

HLT

Before instruction: PC = 0x0100

After instruction: PC = 0x0100

Storage

STO Store
Syntax: STO operand
Description: Stores the contents of the ACC in a memory location specified by the operand.
Opcode: 00001
Status: No bits affected
PC: PC ← PC + 1
Operation: Memory[operand] ← ACC
Example:

STO 0x0002

Before instruction: ACC = 0x0005

MEM[2] = 0x0000

After instruction: ACC = 0x0005

MEM[2] = 0x0005

Procedure Support

CALL Call
Syntax: CALL operand
Description: Performs a procedure call to the memory address specified by the instruction operand field. Before the call, the return address (PC + 1) is stored on the stack (ToS).
Opcode: 11100
Status: Not affected by this operation
PC:

ToS ← PC+1

PC ← operand
Operation: No Operation performed
Example:

CALL 0x00F0

Before instruction: PC = 0x00D3 ToS = 0xXXXX

After instruction: PC = 0x00F0 ToS = 0x00D4

RETURN Return
Syntax: RETURN
Description: Returns from subroutine, retrieving the value stored in the PC stack (ToS). The called procedure must use the ACC to return a value to the calling procedure.
Opcode: 11010
Status: Not affected by this operation
PC: PC ← ToS
Operation: No Operation performed
Example:

RETURN

Before instruction:PC=0x00F0 ToS=0x00D3

ACC=Valor de Retorno

After instruction: PC=0x00D3 ToS=0xXXXX

ACC=Valor de Retorno
